# Image Generation Prompt

Concept visualization only; not fabrication authority.

## Hero Prompt

Photorealistic 3/4 studio product photograph of a small floor-standing harp prototype with a hardwood neck, pillar, base, and spruce-like soundboard, set up as a 22-string diatonic design from low to high strings. Show graduated string lengths, tuning pins along the curved neck, bridge pins or soundboard string path, reinforced neck-pillar-base joints, a stable base footprint, and clean oiled wood finish. The instrument should look prototype-ready and structurally cautious rather than ornate concert-harp mature, on a clean neutral background with soft directional studio lighting, realistic floor scale cues, about a 50mm lens, high detail, photorealistic.

## In-Context Prompt

Photorealistic workshop scene of the floor harp during staged pitch-up, standing on a padded shop floor with a tuner, small ruler, and frame-movement notes nearby. A builder gently tunes one string while staying out of the string plane, natural window light, emphasis on structural load path, tuning pins, pillar angle, and proof-test care.

## Detail / Macro Prompt

Macro photorealistic close-up of the harp neck and tuning-pin field with several graduated strings running toward the soundboard, showing clean pin spacing, smooth string bends, laminated or reinforced hardwood grain, and a small witness mark for frame movement under soft raking light.

## Negative / Avoid

No text, no watermarks, no pedal mechanism, no impossible string crossings, no ornate concert-harp claims, no unsafe body position in the string plane.
