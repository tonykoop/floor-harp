# Floor Harp — Learn to Play

Starter songbook for Tony Koop's floor harp build.

## Tuning

Lever harp tuned to **C major** (all naturals — no lever changes needed for
this songbook). Strings are coloured by convention:

| Colour | Pitch class |
|--------|------------|
| Red | C (all octaves) |
| Black / dark | F (all octaves) |
| White / plain | all other notes |

Tune every string to concert pitch before playing.

## Range

This songbook assumes a **mid-size lever harp** with strings roughly **C3–C6**
(three octaves). If your build has a different range, transpose accordingly.

## Notation

Grand staff: treble clef (right hand) + bass clef (left hand).
All arrangements stay in **C major** — no lever changes required.

## Technique notes

- **Sitting position:** harp tilted back against right shoulder; both arms
  free to reach the strings.
- **Plucking:** use the pads of fingers 1–4 (thumb through ring); pinky is
  not used in standard harp technique.
- **Damping:** after each phrase, lay fingers flat across the strings to stop
  the ring before the next phrase begins.
- **Left hand role:** in these beginner arrangements, left hand plays root
  and fifth on beat 1, then lifts; right hand carries the melody.
